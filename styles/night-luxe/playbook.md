## 1. 风格宣言

1. 这是**深夜的杂志长文**，不是带货短视频——奢华感来自黑暗里的一束光和一行衬线字，不来自金光特效。
2. 画面永远比你想要的**暗一档**：产品像珠宝一样从阴影里浮出来，而不是摆在打光台上。
3. 慢是特权：镜头只做一种运动（缓推/缓移），做完为止；观众的呼吸跟着旁白的句读走。
4. 数字是香槟金的，且必须有口径注文——奢华不豁免出处。
5. 恨：促销感、亮白底、快闪剪辑、"庆祝式"动画，以及一切让它看起来像电商详情页的东西。

## 2. 开拍提问模板

1. 品牌/产品的**视觉母题**是什么？（本片：小棕瓶的琥珀色玻璃 + 夜色）——决定强调色与锚点方向
2. 叙事的**商业转折点**在哪一年/哪件事？——时间轴卡要用
3. 有没有**不可用**的品牌资产（logo/包装是否涉商标风险，是否用"示意重绘"替代）？
4. 大数字卡要压哪 1–2 个数？口径与出处是什么？
5. 收尾姿态：开放式追问 / 冷静收束？（默认冷静收束 + 一句反转）

## 3. 视觉系统

```css
--night:  #0B0E14;   /* 墨蓝夜色（背景基调） */
--velvet: #131722;   /* 丝绒蓝（卡片/分层底） */
--cream:  #EDE6D6;   /* 米白（正文字） */
--gold:   #C9A96A;   /* 香槟金（强调色：数字/关键词/细线） */
--amber:  #8C5A2B;   /* 琥珀（产品母题呼应色，仅点缀） */
--rule:   #2A2F3A;   /* 分隔细线 */
```

- 字体：标题 Noto Serif SC 700/900（衬线，杂志标题感）；正文 Noto Sans SC 400；数字 `tabular-nums` 等宽，金色；英文品牌词用衬线斜体点缀；
- 网格：12 栏，中轴与左对齐混用（开场/收尾中轴，论证段左对齐）；留白 ≥ 30% 画面面积——黑就是留白；
- 层次三级：章节大字 > 论点行 > 注文/出处小字；同屏金色元素 ≤ 2 处；
- style frame 锚点：开拍后产出存 `anchors/`，以"暗场琥珀玻璃 + 一束顶光"为基准帧统一 LUT。

## 4. 节奏与剪辑

- 语速：中文常速 220–260 字/60s；TTS 语气"深夜纪录片旁白"——低沉、克制、句尾不上扬，禁表演腔；
- **声音档（跨片逐字复用，seed-audio-1.0 的 text_prompt 前缀）**：`一位中文男性旁白配音（低沉醇厚，深夜纪录片解说腔，冷静克制、不煽情，语速中慢，句尾平稳不上扬）`；
- 剪辑密度：10–14 cuts/min；切点只落句读；J-cut 默认（画面先到，旁白跟进）；
- 翻译总表引用：`静谧奢华/nocturne`（主行，新增）+ `沉稳/权威`（论证段）+ `干脆/snappy`（仅大数字卡入场覆写：count-up 0.5s，但 ease 用 `power2.out` 禁弹跳）；
- 转场词汇（全 3 种）：硬切（默认）、黑场渐入 0.4s（章节边界）、光斑扫过（仅片头→正文一次）。

## 5. 声部表

> 本包为多来源混排：真实素材（provenance 硬门）与 AI 生成、MG 并置，靠声部语法让切换成为叙事信号。

| 来源 | 角色 |
|---|---|
| 经典电影片段（真实 · **合法免版权源** · provenance 硬门） | 情绪母题 / 档案质感主轨：夜、奢华的氛围锚。来源限公共领域（archive.org/Prelinger）或 CC0（Pexels/Pixabay/Wikimedia）——**禁 YouTube 抓取/版权影视**；SD 档案素材刻意保留颗粒感，不超分成塑料 |
| 产品实拍 / 官方图 | 产品实证位，Ken Burns ≤ 3s，位移 3–5%。真实官方图有商标门；缺料时用生成琥珀瓶锚点（a_style_night）替代，示意不复刻 logo |
| Seedance 生成视频 | 创始年代（1982）AI 复原 + 真实素材缺位时的氛围补镜（真运动，挂锚点） |
| HyperFrames | 论证声部：大数字卡、品牌时间线卡、引言卡、全片字幕层与出处角标 |
| TTS（seed-audio-1.0） | 全片旁白；BGM 低音量夜爵士 / 环境音，-24dB 闪避 |
| 数字人 | 本包不用（faceless 承诺） |

**混排纪律**：真实电影片段与真实产品图必须过 license 硬门（`ledger` 记来源+权利声明），空着不许上；真实素材与 AI 复原之间靠统一 LUT + 共享颗粒缝合，避免"真假打架"。

## 6. 版式 / 镜头模板清单

1. **夜幕开场卡**（3s）：黑场 → 一行衬线钩子渐显 + 金色细线横展；
2. **大数字卡**：金色等宽数字占画面高 50%，count-up 0.5s，下方米白口径注文 + 出处角标；
3. **时间轴卡**：横向金线 + 年份节点，当前节点放大，其余压暗；
4. **引言卡**：衬线引语 + 左侧金色竖条 + 出处小字；
5. **产品 hero 位**（Seedance）：暗场单束顶光，缓推 4–6s，产品居中偏右，负空间留给叠字；
6. **收尾卡**（3s）：论点一句话收束 + 品牌母题剪影 + 数据时效戳。

## 7. 反模式（一票否决）

- 亮白/浅色背景整屏出现；电商促销字体（超粗黑体+描边）；
- 金色滥用（同屏 > 2 处金色即回炉）；数字弹跳 ease；emoji / 图标堆砌；
- 产品镜头不挂锚点直接生成（变脸零容忍）；
- 无出处的数字上屏；亮度一档拉满的"白天感"镜头混入。
