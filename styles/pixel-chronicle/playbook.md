# 风格包 · Pixel Chronicle（像素纪事）

## 0. 档案

- 状态：**候选**（拉片来源：万物电台《佛得角》4′04″，2026-07-16 全片拆解，登记于视频调优工作簿「对标拆解」第 10 行；规则已入《02-知识解说视频风格规范》v0.4。未考片——首片出厂盲评后转正）
- 适用片型：`faceless_explainer` 的人文志/国家志/行业志题；M1 知识科普 3 分钟对标赛道
- 交付承诺类型：`hybrid`（MG 主导 ≈50% + 实拍证据 ≈40%，横屏 16:9——对标片实证知识科普赛道横屏在小红书成立）
- 考片记录：无

## 1. 风格宣言

1. 像一本**被翻旧的图册**：一张纸纹/颗粒底贯穿全片，一切素材（像素 MG、泛黄档案照、实拍、AI 风格化图）都盖同一层做旧——**质感缝合层是五源平权的技术前提**，硬切因此无痕。
2. 数据**实物化**，禁坐标轴图表：薪资对比用钞票堆 + 进度条，人口用人群插画 vs 地标楼群，GDP 涨幅用金色大字 + 像素箭头——观众看的是画，不是图表。
3. 情绪由**剪辑密度**承担，不由音量承担：MG 演示段 10–15s 长镜（内部元素每 2–4s 进出维持视觉变化），情感高潮段 2–3s/镜蒙太奇爆发；口播恒平（-18 LUFS 级），BGM 只垫底。
4. 反差事实连击是默认钩子：首句直接抛最大反差（"GDP 不如两支球队身价"），前 26s 每 2–4s 一个新事实，然后品牌口号闸门 + 黑场开播。
5. 恨：恐吓式剪辑、坐标轴图表、无质感缝合的素材裸拼、情感题材结尾带 CTA。

## 2. 开拍提问模板（elicitation）

1. 讲述对象是什么（国家/行业/事物）？最大的**反差事实**是什么（钩子首句）？
2. 三章递进的骨架：宏观构成 → 历史成因 → 现状结构，末章切**个人叙事**吗？（对标片结构：地理 → 殖民史 → 经济 → 门将个人线 → 情感蒙太奇）
3. 实拍素材从哪来？（授权检索 / 用户投喂 / Seedance 风格化替代——占比 ≈40%，来源必须先落实再开拍）
4. 结尾金句是什么？（情感题材禁 CTA，金句 + 音乐淡出收尾）

## 3. 视觉系统

```css
--paper:   #C9A876;   /* 土黄做旧纸纹底（全片基调） */
--ink:     #2B2118;   /* 深褐（描边/字幕底） */
--accent:  #FFD21E;   /* 亮黄实心（情绪词大字） */
--gilt:    #C9932B;   /* 金色做旧纹理（数字大字） */
--mark:    #C0392B;   /* 红（仅地图箭头/指向，全片唯一冷用途） */
```

- 底层：纸纹 + 胶片颗粒底图贯穿全片，一切素材盖同一层颗粒与褪色 LUT（饱和度压低）；
- 字幕：底部通排**粗宋体白字深描边**，单行 8–16 字，距底 ≈10%，2–3s/屏随口播；关键词不进字幕；
- 大字卡双色分工：亮黄实心 = 情绪词（"生路"/"重返故乡"），金色纹理 = 数字（GDP/530K/10X）；
- 章节卡：暗底 + 品牌字 + 竖排书名号『章节名』+ 圆角白边卡片窗内嵌实拍素材，可承载内容 10–17s；
- 版式四件套位置恒定：左上水印、底部字幕、屏中大字卡、章节卡——模板复用靠质感层救赎。

## 4. 节奏与剪辑

- 语速：口播贯穿型 ~300 字/分（对标实测 304–313），TTS 平实克制零表演腔；
- 翻译总表引用：`沉稳/权威`（MG 演示段主行）+ `紧迫/breaking`（高潮蒙太奇段覆写：2–3s/镜、实拍快切）；
- 结构模板：反差连击钩子（0–26s）→ 品牌口号闸门 + 黑场（~4s）→ 三章递进（章节卡分隔）→ 个人叙事转入 → 情感蒙太奇高潮 → 金句收尾（零 CTA）；
- 转场词汇（全 2 种）：黑场（章节闸门）、**形态转换即转场**（实拍照片逐块像素化溶解成 MG——全片唯一炫技位，≤1 次）；其余全部硬切，质感层保证不跳戏。

## 5. 声部表

| 来源 | 角色 |
|---|---|
| HyperFrames MG | 主轨（≈50%）：像素插画元素进出、大字卡、实物化数据、章节卡、地图 |
| 实拍 / 检索素材 | 证据与情感轨（≈40%）：航拍、市井、人像；高潮蒙太奇必须实拍 |
| 图片动效 | 档案轨：泛黄历史照 + 轻微运动，连续 ≤2 镜 |
| Seedance 生成视频 | **像素叙事主力**（英阿片修订：像素重演/图解由 Seedance 10s 慢组装承担，不只是替补空镜）；生成后必须过缝合层 |
| GPT-Image 静帧 | 像素场景锚点（首尾帧）+ 人物特写像素化；锚点内禁文字数字 |
| TTS | 长片（>120s）**MiniMax speech-2.8-hd 官方音色 single-pass**（实证：`Chinese (Mandarin)_Gentle_Senior`，calm/1.0）；短片可 seed-audio 整片一次。旁白恒平；BGM 垫底不抬升 |
| 数字人 | 本包不用（faceless 片型） |

## 6. 反模式（一票否决）

- 素材不过质感缝合层直接入片（裸拼 = 跳戏）；
- 坐标轴图表（数据必须实物化）；
- 情感题材结尾 CTA（"点赞关注"毁金句）；
- 恐吓式剪辑与音效；
- 高潮段用 MG 凑数（蒙太奇必须实拍质感）；
- AI 风格化画面占比 > 20%；
- 转场特效 > 2 种。

## 7. 秒级结构模板（对标《佛得角》11 镜，2026-07-16 精拆）

> 完整秒级拆解见本目录 `benchmark-breakdown.sec.json`（用户用 Gemini 精拆：11 镜、逐镜机位/构图/色彩/音频、切点表、±0.05s 卡点容差）。复刻新片时按此模板分配时间带，**槽位功能不变、内容置换**：

| 槽位 | 时间带 | 形态 | 功能 |
|---|---|---|---|
| S1 | 0–22s | 像素 MG 数据连击 | 反差事实钩子（图形随旁白逐个弹出，卡点对齐） |
| S2 | 22–34s | 实拍 → 3D/像素地图 | 品牌口号闸门 + 报题 + 地理定位 |
| S3 | 34–56s | 像素 MG 图解 | 章一·对象的"构成"（体感对比锚：饼干屑/天通苑式） |
| S4 | 56–84s | 实拍空镜 → 像素历史 | 章二·历史成因（情绪转沉重，实拍→像素硬切） |
| S5 | 84–100s | 纯实拍纪实 | 现实困境（无文字、无特效——让素材自己说话） |
| S6 | 100–125s | 像素+实拍交替 | 章三·结构分析（分流图示 + 实拍佐证） |
| S7 | 125–141s | 实拍 + 像素点缀 | 转折铺垫（环境改善/秩序） |
| S8 | 141–164s | 像素卡片流 + 情绪角色 | 递进到情绪峰前站（卡片依次滑出 + 角色表情转变） |
| S9 | 164–187s | 像素动作高潮 | 主叙事高潮（体育/冲突的像素重演——**版权画面在此槽位用像素重演规避**） |
| S10 | 187–200s | 实拍真人特写 | 情感落地（唯一真人面孔特写位，像素回到实拍=情绪着地） |
| S11 | 200–243s | 实拍蒙太奇 | 收尾狂欢/致意 + 手写体金句 + 淡出（零 CTA） |

模板级硬规则（从精拆新提炼）：**MG 段与实拍段严格交替**（S1像素→S2实拍→S3像素→S4混→S5实拍…），交替本身就是节奏；S1 的图形弹出必须与旁白逐词卡点（±0.05s 容差）；SFX 层不可省（UI 弹出音/金币声/落物声——像素段的"体感"一半来自音效）；S5 纪实段禁一切文字与特效；S10 特写位只留一张真实面孔。

## 9. 定稿工艺（英阿《什么仇什么怨》v1→v6 六版实战沉淀，2026-07-19 — 复刻新片直接照此执行）

> 目标：新片**首版即达 v5 验收质量**，不再靠用户五轮返工教。每条都有引擎知识包对应展开（括号内），本节是风格包视角的汇总。

1. **音频先行链**：MiniMax 官方音色 single-pass 整条 → wav2vec2 强制对齐逐字戳 → 字幕/画面 cue/挂载全部由同一 timeline 派生（`tts-audio.md` `forced-alignment.md`）。compose 生成器里**零手写秒数**——改文案/换音色只重跑链条，版式不用动。
2. **像素镜头**：Seedance 10s 慢组装（5s 会被打回"太快"）、挂载尾对齐让 payoff 帧落在文案字上、锚点静帧零文字（数字走 HTML 角标）（`seedance.md`）。像素叙事覆盖 ≥40%：钩子事件、历史重演、地图/距离/贸易等**强叙事位优先像素**，检索素材只作证据与氛围。
3. **实拍素材分层**：Pexels 只做纯空镜；叙事位走 archive.org/Commons 公域档案；时事位走 APIhub 新闻（broadcast-risk ≤3s 闪切+告知用户）（`footage-sourcing.md`）。
4. **空镜池五铁律**：一素材全片一次（断言）、关键素材按段 RESERVE、时代/地点错位过滤、素材窗尾留 0.8s 防暗尾、轨道零缝隙（`footage-sourcing.md`）。
5. **BGM 渲染前挂入** composition 独立轨：纪录片底色 + 轻 chiptune、LRA ≤ 5、比旁白低 ~20 LU（-30 LUFS 素材挂 `data-volume 0.23` 级）、时长按 VO 自适应拼接（交叉淡化 3s）。成片 `loudnorm I=-14:TP=-1.5`。
6. **虚实呼应手法（本包签名动作）**：同一意象"像素重演开场 + 真实画面收尾"（英阿片：像素横幅拉开 @钩子 ↔ 真实横幅新闻 @"横幅在亚特兰大的夜里拉开"）——蒙太奇并置替代转场炫技。
7. **出厂评估**（⑨ review 必过，`visual-selfcheck.md` 硬查 1–15）：素材零重复、时代错位、暗尾黑闪、音画 6 锚点抽查、**时效词复核**（相对时间词逐个对绝对日期核算）、烘焙镜头拍点。抽帧 contact sheet 两轮（整数点+半步错位）+ 定点帧。
8. **迭代纪律**：用户每轮反馈先归因到"根因层"（对齐器数学/池路由逻辑/引擎上限），修机制不修表象；机制修完立刻沉淀回知识包——这是本节自身的由来。
