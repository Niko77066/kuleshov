# G08 表层行文画像：不是“有梗”，是拒绝模型胶水

> 作用：在完成材料图与段落接力后，控制表层行文。它学习的是用户提供原文中可抽象的句法、视角、信息压缩与转弯方式；不复用原句、固定口令、署名人格或可识别的个人声音。

## 1. 原文的表层发动机

### 1.1 不是中立解释者，而是带着判断下场的人

叙述者先把一个细节当作很重要的线索；对它作带偏见、但可追踪的解释；再让这个解释把下一个具体对象拖进来。不是先给“客观框架”，最后才给态度。

可迁移控制：每段必须至少有一个可指认对象、一个动作、一个主观判断；判断必须压在前述材料上，不能写成空泛的“值得关注”。

### 1.2 不用对称句把道理熨平

原文允许句子偏、断、先说过头再往回收。它追求的是“这个人为什么突然想到这件事”，不是“读者是否被完整地照顾”。

可迁移控制：一段先按事件的奇怪顺序走，再在段内补解释；不得每段都按“观点 → 解释 → 生活类比 → 温和结论”四步齐全。

### 1.3 笑点是材料的二次结论，不是比喻库

原文的荒诞来自专名、旧关系、原话中的限定词、具体数字和日期碰撞；笑点出现后还会成为下一段证据。它不是看见“行业”就自动匹配地铁、外卖、超市或考试。

可迁移控制：一个笑点若无法回指事件中的专名/动作/原话/数字，删掉。泛生活类比一稿最多一个，且它必须在后文再次承担推理。

### 1.4 反转要付材料费

“你以为……其实……”本身不高级；它只有在前面已经认真建立一个可相信的答案、随后又有新事实把答案掀翻时才有效。第 8 篇里，候选赢家不是作者随手换的标签，而是上一段泄露出的关系或约束。

可迁移控制：每次反转都必须标注 `先前承诺 / 新事实 / 为什么此刻才知道`。缺任一项，禁止使用反转句式。

## 2. 高风险 AI 胶水：默认禁用，满足条件才可解锁

| 胶水句式 | 为什么像 AI | 解锁条件 |
|---|---|---|
| “不是 A，而是 B” | 把复杂关系偷压成标准答案 | A 与 B 都是本段刚出现的具体对象或动作；B 必须被新事实证明，不能是抽象结论。 |
| “你以为……其实……” | 预制反转，不需要材料 | 前段已用完整事实让“你以为”成立；“其实”带来下一节点，且不超过全稿一次。 |
| “从表面看”“说白了”“归根结底”“这说明了” | 作者替观众抹平推理 | 一律不写；让对象、动作和后果自己说明。 |
| “真正的赢家/答案是” | 提前宣布 thesis，候选链失效 | 仅能出现在母稿原本揭晓位置；此前不能出现同义结论。 |
| “对观众来说”“最终落到普通人” | 强行普世化和道德收束 | 只可由开场已经出现的具体普通动作回收；禁止新开“观众/消费者”分类段。 |

这些词不是永久黑名单。问题不是词，而是模型把它们当作无成本连接件；每一次使用都必须由材料付费。

## 3. G08 的表层写作约束

1. **先专名，后类别。** 先写谁做了什么，再允许说“平台”“资本”“饭圈”一类概念。
2. **先荒诞细节，后解释。** 解释最多连续两句，第三句必须增加新动作、旧关系或能把话题拐走的事实。
3. **不替观众做总结。** 删去“这告诉我们”“所以我们应该”“对普通人而言”；结尾只回收开篇已经出现的冲突或物件。
4. **允许不匀称。** 不强迫每段都有比喻、反方、价值观与落点；母稿的段落本来就有信息密段和纯转场段。
5. **一句话只活在这个事件里。** 删除专名后仍能原封不动放进任何商业文章的句子，视为 AI 句，重写。
6. **态度必须具体。** 不写“有趣、复杂、值得思考、反映了”；写清楚哪一个动作荒唐、谁在装糊涂、哪一笔账被谁藏起来。

## 4. 出厂检查：表层不是最后润色，而是硬门

在 Anti-Smooth Gate 之后逐段标记：

- `named-action`：本段的专名 + 动作；
- `stance`：叙述者究竟站在什么判断上；
- `glue`：是否使用高风险胶水；若用，填解锁事实；
- `generic-sentence`：删专名后是否仍能放进别的稿；若能，必须删改；
- `callback`：本段的笑点或物件是否会在后文承担新作用。

任一段 `generic-sentence = true`，或任一高风险胶水没有解锁事实，整稿不得进入 TTS。
