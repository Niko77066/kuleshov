# 风格包 · Case-File 案卷（官号实证快报）

> 名字可改。定位：**官号运营（企业/政务/机构号）的新闻·政策·事件解读**，竖屏 9:16。把 **MG 动画 + AI 纸拼贴 b-roll + 文件图片** 三语言组合固化下来。

## 0. 档案

- 状态：**首考通过 2026-07-17**（用户认可「这个风格包应该就没问题了」；首片 `openai-78m-logs` GS-11 · 4.22/5）。替代已淘汰的 `daily-brief`。
- 交付承诺类型：`typography_led + collage_broll + document`（三语言混排）；**时长无锁**（音频/内容自然定）。
- 由来：daily-brief 首片被用户判「全军覆没」（PPT 味 / 上堆下空 / 提文件却在摆字）。本包是那次尸检后的重构，吸收 samsung `tech-newsroom` 的文件精读技法 + `collage-broll` 声部 + 两个幸存镜（居中极简）的经验。

## 1. 风格宣言（每条都对着 daily-brief 的死因）

1. **视觉重心居中**。竖屏用户视线在画面中央——内容填满中部，**严禁上堆下空**（daily-brief 死因之一）。
2. **层级极简**。每镜 ≤ 2 个信息元素；**砍掉「左上 kicker + 左下 src 小字」那套双角标**（PPT 味的直接来源）。出处并进画面、或压成一条极轻脚注，不抢视线。
3. **文件前置，不堆字**。提到具体文件/法规/数据，就**带观众读文件**：真截图（能拿到）或高保真复现（拿不到）——浏览器 chrome + 真实来源 URL + 逐字真实文本 + **随旁白同步高亮**。杜绝「纯文字版式卡讲文件」。
4. **数字压在文件上**。关键数字用大 count-up / 高亮 callout **叠在文件图上**，不做孤立数字卡。
5. **拼贴作隐喻间奏**。抽象/情绪/概念镜走 AI 纸拼贴 b-roll，与文件/版式硬切成节奏。
6. 恨：PPT 双角标、上堆下空、纯版式讲文件、孤立数字卡、每镜换版式、彩虹色、emoji。

## 2. 三声部分工（本包立身之本）

| 声部 | 表达什么 | 知识包 |
|---|---|---|
| **文件图片 document** | 法规/起诉文件/报道/后台——实证。浏览器 chrome + 同步高亮 | 本文 §5 + samsung 参照 |
| **AI 纸拼贴 b-roll** | 概念/隐喻/氛围间奏 | `produce/references/collage-broll.md` |
| **MG 版式 HyperFrames** | 钩子/论点/收尾/大数字 count-up——**居中极简** | `produce/references/hyperframes.md` |

## 3. 视觉系统

- 色：纸白 `#F4F1EA` / 墨黑 `#17181C` / **印刷红 `#C8451B`**（强调）/ 细线 `#C9C4B8`。品牌色可覆写。
- 版式卡：**内容垂直居中**（`justify-content:center`）；单屏 ≤ 2 元素；超大字（主标题 120–184px）；**无角标杂字**。
- 字体：报头衬线 `Songti SC`；正文 `PingFang SC`；数字 `tabular-nums`。
- 文件 chrome：浏览器框（三色点 + 地址栏真 URL）或公文头；圆角 + 柔阴影；真实文本。
- 品牌：右下角 logo，低透明，收尾放大；**不做常驻角标小字堆**。

## 4. 节奏与剪辑

- 居中大字入场：`power4.out` 0.4s；数字 count-up ≤ 0.6s，`tabular-nums` 防抖。
- 文件精读：开场特写（scale 1.6–1.9）→ 0.8s 展开全览 → 极缓上移（`none` ease）保活性。
- 转场 ≤ 4：硬切（默认）/ 色块横扫（章节）/ 白帧（收尾前）。
- 拼贴 ↔ 文件 ↔ 版式 之间硬切，靠内容推进、不靠花转场。

## 5. 文件精读技法（samsung 式，本包核心）

1. **真截图优先**；拿不到（gov 站 / 付费墙 / 浏览器拦截）则**高保真复现**：浏览器 chrome + 真来源 URL（地址栏）+ **逐字真实文本**（必须准确、可回链 research）+ 公文/文章样式。复现是**可视化不是伪造**：文本准确、来源标注、editorial 呈现。
2. 浏览器 chrome：三色点 + 地址栏（如 `cac.gov.cn` / `techcrunch.com`）。
3. 运动：开场标题特写 → 0.8s 展开 → 极缓上移保活性（参 samsung `s03-consent-read.html`）。
4. **同步高亮**：讲到哪一行/哪个数字，那里 wash（淡色块 opacity 0→1）+ bar（scaleX 0→1，印刷红）按旁白**词级时间**落。
5. 数字 callout：大数字叠在文件相关处，count-up + 印刷红下划/框。

## 6. 反模式（一票否决 · daily-brief 尸检清单）

- 左上 kicker + 左下 src 的双角标（PPT 味）；
- 内容上堆、下半屏大留白（竖屏视觉重心错位）；
- 纯文字版式卡讲「具体文件」（该读文件却在摆字）；
- 孤立数字卡（数字该压在文件上）；
- 每镜换版式 / 彩虹色 / emoji。
