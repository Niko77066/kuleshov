# 引擎知识包 · 实拍/检索素材：来源分层与空镜池纪律

> 何时读我：storyboard 有镜头路由到"实拍/检索素材"源；compose 组装空镜池前（**必读**）。
> 由来：英阿片 v2→v5 素材三轮返工沉淀（2026-07-17/18 用户逐条验收）：相关性差、重复率高、时代错位、黑屏暗尾，每条都吃过打回。

## 来源分层（用户拍板，2026-07-17）

| 层 | 来源 | 只准干什么 |
|---|---|---|
| **纯空镜/氛围位** | Pexels / Pixabay | 海面、天空、城市泛景、球场灯光、草皮特写——**不承担叙事** |
| **叙事/证据位** | archive.org + Wikimedia Commons **公有领域档案** | 历史事件的真画面：老纪录片、版画/档案照（静帧缓推）、战争档案 |
| **时事/新闻位** | **APIhub**（`youtube_search → video_info → video_download`）官方新闻频道 | 近期事件的新闻画面 |

- **叙事素材绝不许用泛用空镜顶替**（"德式老城冒充布宜诺斯艾利斯、集装箱船冒充 1900s 贸易"= v3 反例存档）。检索不到就记 `manifest.gaps`，宁缺——**禁静默顶替**。
- 版权分级记 manifest `license` 字段：public-domain / CC（注明变体）/ pexels / **broadcast-risk**（新闻转播类：单条使用 ≤3s 闪切 + 交付时明示用户拍板）。
- 工程坑：Pexels 检索必带 UA、下载必带 `-A UA -e https://www.pexels.com/`；APIhub 是 MCP over HTTP（initialize 拿 `Mcp-Session-Id`，SSE 响应）；new-api 网关封 python-urllib UA（一律 curl）。

## manifest 契约（`compose/assets/brollN/manifest.json`）

每条：`{file, source(url), license, section, desc(中文一句话画面内容), dur_s}` + 顶层 `gaps[]`、`keep_from_broll2[]`（跨版保留项带 reason）。**desc 是池路由的唯一依据，必须写实**（时代、地点、内容物）。

## 空镜池组装五铁律（compose 生成器强制执行，全部可断言）

1. **一素材全片一次（零容忍）**：单素材全片最多出现一次。宁可把切段从 3s 放宽到 5s，也不重复画面。构建后断言：`Counter(srcs) 无 >1`。池耗尽走 fallback 链（tag 过滤 → 同 section 未过滤(带 era 排除) → 任意未用），仍不够**大声报错**，禁静默循环。
2. **关键素材按段预留（RESERVE）**：叙事强锚素材（如"墨西哥城"必须配阿兹特克航拍、金句收尾必须配夜景、"横幅拉开"必须配横幅新闻）在生成器里**显式预留给对应段**，防止被前面的段落 fallback 连锁挪用（v5 实翻过：温布利段借走阿兹特克 → 墨西哥城配上了布宜诺斯艾利斯方尖碑）。
3. **时代/地点错位过滤**：池按 desc 关键词 include（`TAG_FILTER`）+ exclude（`TAG_EXCLUDE`，如 1806 段禁"伦敦/街景/银行"）——1806 的叙事段出现 1920s 汽车街景 = fail。
4. **素材窗尾留余量防暗尾黑闪**：挂载窗 `media_start + dur ≤ 素材时长 − 0.8s`——很多素材末尾有淡出/暗场，取到尾部 = 观感"莫名黑屏"（v4 反例）。
5. **轨道零缝隙**：构建后断言主轨相邻 clip 无重叠、无 >0.35s 缝隙（章节卡/照片盖住的除外）；<1.3s 的小缝自动并入前段。

## 切段节奏

- 单段 2.2–5s（默认目标 3–4s），段内容量不足时并入前段，不留 <1.2s 碎尾；
- 池内排序按 desc 命中偏好词（`TAG_PREF`），让最贴题的素材先上；
- 重编码规格统一：1280x720（crop 填满）、h264、`-g 12 -keyint_min 12 -sc_threshold 0`、`-an`、观感与既有池一致（必要时 `eq=saturation=0.8` 轻做旧——最终统一 LUT 在 compose 层）。

## QC（下载后 & 成片后各一轮）

- 下载后：抽 3 帧人工看图（模糊/大水印/摆拍/不贴题→丢弃，记废弃理由）；
- 成片后：contact sheet 两轮（整数点 + 半步错位）逐帧过——重点抓：时代错位、池借用错位、暗尾黑屏、素材重复既视感。
