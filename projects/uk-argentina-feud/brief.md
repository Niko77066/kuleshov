## pixel-chronicle 开拍四问（elicitation）

1. **讲述对象与最大反差事实**：候选钩子——"为了几座企鹅比人多的岛，两个国家真刀真枪打了 74 天"（数字待 research 核实）；或挂时事——"明天这场球，是两个国家 X 年恩怨的最新一章"。script 阶段定稿。
2. **三章递进骨架（预案）**：①钱的恩怨——19 世纪英国资本控制阿根廷铁路与牛肉贸易（"非正式帝国"），阿根廷曾是世界最富国家之一 → ②血的恩怨——1982 马岛战争 74 天，贝尔格拉诺号，两国政治命运反转 → ③球的恩怨——1966 拉廷事件、1986"上帝之手 + 世纪进球"（马拉多纳的"报复"叙事）→ 收尾转入 2026 这场球 + 个人叙事情感蒙太奇（对标片结构复刻）。
3. **实拍素材来源**：历史档案照丰富（战争影像、86 年比赛照）——图片动效声部主扛；**足球比赛版权画面（FIFA）严禁直接使用**——用像素 MG 重演"上帝之手"瞬间（pixel-chronicle 的像素画天然规避版权 + 风格化，这是本包最大优势）；现代空镜（布宜诺斯艾利斯/伦敦/马岛）待检索 API 或 Seedance 风格化兜底。
4. **结尾金句**：script 阶段出（方向：战争没有赢家，但球场给了两国一个"文明算账"的地方——情感题材禁 CTA，金句 + 淡出收尾）。

## 风险与纪律

- **时效锚是硬依赖**：research 必须核实英阿是否真有对阵、几点开球——发布须踩在赛前/赛后热度窗内；若无对阵则确认文章由头是什么，钩子改挂真实热点。
- 战争题材守 pixel-chronicle 反模式：**禁残骸/伤亡画面、禁恐吓剪辑**——张力来自"看懂恩怨链条"，不是来自血。

- 数字全部带出处与对比锚（74 天、伤亡数、距离对比：马岛离阿根廷 vs 离英国）。
